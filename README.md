# 一系列平行直线拟合

已知直线斜率相同，故可先采用单条直线拟合的算法求出拟合的斜率.

确定斜率后即可算出对应各点截距，问题转化为一维的截距拟合.

OLS 对应结果即为均值，RANSAC 只需线性扫描即可.
